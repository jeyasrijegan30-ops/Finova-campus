
# Finova Campus – College Finance Management System

## Project Overview

Finova Campus is a web-based finance management application developed for educational institutions. The system helps manage and organize financial records efficiently through a simple and user-friendly interface.

## Features

* Financial record management
* Income and expense tracking
* Organized data management
* User-friendly dashboard

## Technologies Used

* HTML
* CSS
* JavaScript

## Purpose

The project was developed to simplify financial data management and improve accessibility of financial records within educational institutions.
